  1. Add the following code at the end of your HTML body tag:
  2. Place a div with the frik-main ID (you can include HTML code before and after this div for apply it on all pages):
  3. Create some HTML pages in a subdir pages in the dir of your main HTML code. They need to follow this structure: 
    
    - index.html
      - /pages
      - main.html (the default page, who is diplayed as the homepage)
      - random_page.html (you can create a infinite number of these pages if you want!)
  4. For make a link, just add the following in a a or button tag: 
    
    data-href="/**the path to your html page, without the *pages* at the start**"

Options

You can pass optionnals options to Frik object initiator for customize the router:

afterRender The function to call after render of a page (to write without the () at the end)
afterRender-args The args to pass to the function of afterRender option
loadingHTML Custom HTML to use instead of Loading... text
